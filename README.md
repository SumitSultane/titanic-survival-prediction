# 🚢 Titanic Survival Prediction using Machine Learning

## 📌 Project Overview
This project aims to predict whether a passenger survived the Titanic disaster using Machine Learning techniques. It follows a complete data science workflow including data preprocessing, visualization, feature engineering, model building, and evaluation.

---

## 🎯 Objective
To understand and implement the complete data science pipeline using a real-world dataset and build a predictive model.

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 📊 Dataset
The dataset used is the Titanic dataset which contains passenger details such as:
- Age  
- Gender  
- Passenger Class (Pclass)  
- Fare  
- Family details  

---

## 🔄 Steps Performed
1. **Data Collection**  
   Loaded the Titanic dataset  

2. **Data Cleaning**  
   - Handled missing values  
   - Removed unnecessary columns  

3. **Data Visualization**  
   - Survival Count Plot  
   - Age Distribution  
   - Correlation Heatmap  

4. **Feature Engineering**  
   Created a new feature called **FamilySize**

5. **Model Building**  
   Applied **Logistic Regression** algorithm  

6. **Model Evaluation**  
   - Accuracy Score  
   - Confusion Matrix  

---

## 📈 Results
- Model Accuracy: **81%**  
- The model successfully predicts survival with good performance
